Web Designer Manami Sampei

さいたま在住のWebデザイナー、三瓶まなみです。オンラインスクールでWeb制作について学び、ハウスメーカー勤務時から興味があったWebデザイナーへと転職活動中です。まだ未熟な部分もありますが、これからさらにWeb制作の技術や知識を身につけてWebサイト運用や問題解決のアドバイスもできるデザイナーになりたいと思っています。

Skills

Web

最低限のHTML/CSSでのコーディングができます。また、JavaScript、PHP、WordPressも並行して勉強しています。

Text

文章を書くことが好きです。前職では「どうすれば分かりやすく伝わるか」を考えて、自社サイトの更新やカタログ作成を行っていました。

Talk
